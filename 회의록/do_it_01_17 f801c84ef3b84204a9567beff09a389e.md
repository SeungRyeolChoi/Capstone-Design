## Github Organization

- 레포지토에 대한 공동 접근 권한을 가진다.
- 팀 멤버의 역할과 권한을 관리할 수 있어 프로젝트의 보안과 협의 효율성 up
- 권한 관리 쉬움. 위임할때도 쉬움
- 이슈트레킹 및 pull request 관리 용이
